# **App Name**: Skincare Savior

## Core Features:

- Product Filtering: Allow users to filter products based on skin type, skin concern, price, brand, and product category.
- Skin Quiz: A short quiz to determine skin condition, dryness/oiliness, sensitivity, and main needs, outputting 3â€“5 matching products.
- Product Details: Display product name, short description, ingredients, functions, targeted results, suitable skin types, and usage instructions.
- Wishlist: Allow users to save products to a wishlist for future purchase.
- Shopping Cart: Enable users to add, modify quantities, and remove products from their shopping cart.
- AI Skin Advisor: A tool that will suggest appropriate skincare products for oily skin based on user input.
- Voucher Program: Allow users to upload a photo of empty packaging, wait for admin approval, and receive an automatic voucher in their account. Display voucher history with status and expiration.

## Style Guidelines:

- Primary color: Soft Lavender (#E6E6FA) to evoke a sense of calm and skincare.
- Background color: Very light off-white (#F8F8FF), almost imperceptible, for a clean, modern feel.
- Accent color: Pale pink (#FFB6C1) for interactive elements, providing a subtle contrast.
- Body and headline font: 'Alegreya', a serif font with an elegant and intellectual feel.
- Use simple, clean line icons for product categories and UI elements.
- Ensure a clean, spacious layout with clear hierarchy, emphasizing product visuals.
- Use subtle animations on hover and transitions to provide user feedback and enhance the experience.
